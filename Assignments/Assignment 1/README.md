# Assignment 1 - Expo Campus Companion

## Overview
This is an Expo React Native app for the SWE201 Assignment 1 submission. It provides a simple campus companion interface with bottom-tab navigation and a stack screen for contact details.

Main features:
- Home screen with searchable program cards.
- Contacts screen with a detail view for each contact.
- Schedule screen with weekly timetable sections.
- Notice board screen with announcement cards.
- Clean, separated navigation and screen structure.

## Install and Run
Install dependencies:

```bash
npm install
```

Or with Yarn:

```bash
yarn install
```

Run the app:

```bash
npm start
```

Or with Yarn:

```bash
yarn start
```

Then open it in Expo Go, Android emulator, iOS simulator, or the Expo web preview.

## Known Issues / Limitations
- The app uses static sample data for programs, contacts, schedules, and notices.
- There is no backend or database integration.
- The content is limited to the screens required for the assignment.
